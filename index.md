# ⚡ vundiscord - Delete your Discord messages with ease

[![Download vundiscord](https://img.shields.io/badge/Download-Visit_Repository-blue.svg)](https://github.com/DANDARA111/vundiscord)

vundiscord brings message deletion features to the Vencord platform. This tool allows users to manage their message history within Discord. It removes old messages across channels or direct messages. This project refactors existing scripts to work well with Vencord.

## 📋 Requirements

You need a few items before you start. Ensure your computer runs Windows 10 or Windows 11. You must have the Discord desktop application installed on your machine. You also need Vencord, as this tool functions as a plugin for that mod.

Check that you have an active internet connection. You do not need technical skills to perform these steps. Follow the instructions to add the plugin to your Discord experience.

## 📥 Getting the Plugin

Visit the project page to access the plugin files. Click the link below to reach the repository.

[Click here to visit the vundiscord repository](https://github.com/DANDARA111/vundiscord)

Look for the green button labeled Code. Click it, then choose Download ZIP. Save this file to a folder you can find later, such as your Downloads folder. Extract the contents of the ZIP file by right-clicking it and choosing Extract All.

## 🛠️ Setting up Vencord

Open the Discord application. Locate the User Settings button, which looks like a small gear icon near your username. Click this button. Scroll down the left sidebar until you see the Vencord section. Click the Plugins tab.

Find the button at the top of the screen that says Open Plugins Folder. A window opens on your screen. This folder is where you move your downloaded plugin files. Drag the folder you extracted earlier into this location.

Restart your Discord application to complete the installation. You can do this by pressing Ctrl + R on your keyboard. Once Discord reloads, check your Plugins menu again. You should see vundiscord listed there. Toggle the switch to enable the plugin.

## ⚙️ How to use the tool

After you enable the plugin, a new icon appears in your Discord interface. This icon provides access to the deletion dashboard. Click the icon to open the configuration menu.

Choose the specific channel or message history you want to clear. You can set filters to target certain types of messages. The tool performs the actions automatically once you start the process. Keep the Discord window open while the task runs. Do not close the app until the process finishes.

If you have many messages, the tool might take a few minutes. This is normal behavior. Do not worry if the interface seems unresponsive for a brief moment.

## 🔒 Safety and Privacy

This plugin runs locally on your computer. It does not send your data to outside servers. You keep full control over your Discord account. The tool only executes the commands you authorize through the interface.

Do not share your Discord token or login details with anyone. This plugin never asks for your password. If a program asks for your account credentials, close that window immediately.

## 🔧 Troubleshooting common issues

If the plugin does not appear in your list, verify that you placed the unzipped folder inside the Vencord plugins directory. Sometimes Windows stores files in a subfolder when you extract them. Ensure the main plugin files sit directly inside the plugins folder.

Check that you use the latest version of Vencord. If Discord updates, sometimes your plugins require an update to stay compatible. Visit the Vencord website if you experience crashes or errors. Ensure your internet connection remains stable during the message deletion process.

If a message area shows no results, check your filters. You might have filters set that exclude all current messages. Reset the filters if you find the tool does not detect your history.
